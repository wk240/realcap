> 调研日期：2026-04-28
> 目的：分析目前依赖截图验证工作真实性的行业场景，评估RealCap产品的目标市场

---

## 一、金融信贷领域

| 场景 | 验证内容 | 甲方 | 乙方 | 伪造风险 |
|------|---------|------|------|---------|
| 房贷申请 | 银行流水、存款余额 | 银行/房贷机构 |购房者 |⭐⭐⭐ 高 |
| 车贷申请 | 收入证明、银行流水 |汽车金融公司 |购车者 | ⭐⭐⭐ 高 |
| 信用卡申请 | 收入证明、资产证明 | 银行 |申请人 | ⭐⭐ 中 |
| 小额贷款 | 支付宝/微信余额 |网贷平台 |借款人 | ⭐⭐⭐⭐极高 |
| 消费分期 | 收入证明 |分期平台 |消费者 | ⭐⭐⭐ 高 |

**现状**：
- 大银行：要求纸质流水或API对接，不接受截图
- 小机构/网贷：接受截图，伪造泛滥

## 按伪造风险分级

### 高风险场景（伪造需求大）

| 场景 | 为什么伪造需求大 |
|------|-----------------|
| 小额贷款 | 借款人急需资金，伪造余额骗取放款 |
|相亲/社交 | 用户想证明"有钱"进入高端圈子 |
| 直播/MCN签约 | 网红伪造粉丝数/收入骗取签约金 |
| 助学金/低保 | 反向伪造：证明收入"低"骗取补贴 |
| 游戏账号交易 | 伪造虚拟资产骗取买家付款 |

### 中等风险场景

| 场景 | 说明 |
|------|------|
|租房申请 | 租客伪造收入证明租到好房子 |
| 车贷/房贷 | 部分小机构接受截图 |
| 背景调查 | 求职者伪造过往收入 |

### 低风险场景

| 场景 | 说明 |
|------|------|
| 大银行房贷 | 不接受截图，要求纸质流水 |
|正规背调机构 | 有专业验证流程 |
| 正规保险理赔 | 要求纸质凭证 |

## RealCap目标行业建议

### 最有价值的目标行业

| 优先级 | 行业 | 理由 |
|--------|------|------|
| ⭐⭐⭐⭐⭐ | 小额贷款/网贷 | 伪造泛滥，平台损失大，付费意愿高 |
| ⭐⭐⭐⭐⭐ | 直播/MCN签约 | 伪造粉丝数/收入，机构损失大 |
| ⭐⭐⭐⭐ |相亲/社交平台 | 用户愿意付费证明真实性 |
| ⭐⭐⭐⭐ | 游戏账号交易 | 虚拟资产交易依赖截图，伪造泛滥 |
| ⭐⭐⭐ |租房中介 | 中介机构有批量验证需求 |

### 不建议作为目标的行业

| 行业 | 理由 |
|------|------|
| 大银行房贷/车贷 | 不接受截图，已有API对接 |
| 正规保险理赔 | 要求纸质凭证 |
| 政务/低保/助学金 | 需要官方渠道证明，截图不被认可 |

## 目标客户分析

### 甲方侧（付费方）

| 客户类型 | 验证需求 | 付费意愿 |
|----------|---------|---------|
| 网贷平台 | 验证借款人资产/收入 | 高（减少坏账损失） |
| MCN机构 | 验证网红粉丝数/收入数据 | 高（避免签约欺诈） |
| 相亲/社交平台 | 验证用户身份/资产真实性 | 中（提升平台可信度） |
| 游戏交易平台 | 验证虚拟资产真实性 | 高（减少交易纠纷） |
| 租房中介 | 验证租客支付能力 | 中（批量验证需求） |

### 乙方侧（用户）

| 用户类型 | 验证动机 | 配合意愿 |
|----------|---------|---------|
| 借款人 | 获取贷款 | 高（需要放款） |
| 网红/主播 | 签约MCN | 高（需要签约） |
| 相亲用户 | 进入高端圈子 | 高（证明真实性） |
| 游戏卖家 | 出售账号 | 高（完成交易） |
| 租客 | 租到房子 | 中（配合房东要求） |

---

## 结论

RealCap最有价值的目标客户组合：

**最佳切入点**：
1. **小额贷款/网贷平台** — 伪造风险最高，甲方付费意愿最强
2. **MCN机构** — 验证网红数据，避免签约欺诈损失
3. **游戏交易平台** — 虚拟资产依赖截图验证，市场活跃

**后续拓展**：
4.相亲/社交平台 — 用户侧付费意愿可能更高
5. 租房中介 — 批量验证需求，但付费意愿中等

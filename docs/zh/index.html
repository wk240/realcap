Common Questions

什么是截图验证？

截图验证确认截图是真实拍摄的，未被篡改或由AI生成。RealCap在截图瞬间创建不可篡改的真实性记录。

RealCap如何检测AI生成的虚假截图？

RealCap在截图瞬间捕获元数据，包括设备信息、时间戳和截图上下文。这些元数据经过加密，无法事后添加到AI生成的图像中。

RealCap可以验证安装前拍摄的截图吗？

不可以。RealCap只能验证通过我们的应用、浏览器扩展或API拍摄的截图。安装RealCap前拍摄的截图无法事后验证。

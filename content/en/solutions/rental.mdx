# Rental Platform Screenshot Verification

Rental platforms face serious financial and identity screenshot fraud problems when reviewing tenant applications. RealCap provides a professional screenshot verification solution to help platforms verify tenant income proof and identity document authenticity, protecting landlords from bad tenant losses.

## Industry Pain Points: Tenant Fraud Problem

Rental markets involve large amounts of financial proof and identity verification, these key documents facing serious threats from AI forgery technology.

### Common Fraud Types

Common screenshot fraud types in rental applications:

**Income Proof Fraud:**

| Type | Description | Purpose |
|------|-------------|---------|
| AI-generated paycheck | AI generates fake paycheck screenshots | Forging high income for premium properties |
| Modifying real income proof | Photoshop modifying income amount | Increasing income for rent discounts |
| Forging freelance income | Fabricating fake income screenshots | Hiding real low-income status |
| Using others' income proof | Impersonating using others' income screenshots | Impersonating identity for properties |

**Bank Balance Fraud:**

- AI generating fake bank balance screenshots
- Modifying real bank balance numbers
- Using others' bank screenshots impersonating
- Forging deposit certificates

**Identity Document Fraud:**

- AI generating fake ID documents
- Modifying real document information
- Using others' documents impersonating for rental
- Forging work certificates

### API Integration Example

```javascript
// Rental platform verification integration example
async function verifyTenantApplication(applicationId, documents) {
  const results = {};
  
  // Income proof verification
  if (documents.incomeProof) {
    const incomeResult = await fetch('https://api.realcap.app/verify', {
      method: 'POST',
      headers: {
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        verificationType: 'rental_income',
        applicationId: applicationId,
        screenshot: documents.incomeProof,
        metadata: {
          declaredIncome: documents.declaredIncome,
          declaredEmployer: documents.declaredEmployer,
          propertyRent: documents.propertyRent
        }
      })
    });
    results.income = await incomeResult.json();
  }
  
  // Bank balance verification
  if (documents.bankBalance) {
    const balanceResult = await fetch('https://api.realcap.app/verify', {
      method: 'POST',
      headers: {
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        verificationType: 'rental_balance',
        applicationId: applicationId,
        screenshot: documents.bankBalance,
        metadata: {
          declaredBalance: documents.declaredBalance,
          propertyRent: documents.propertyRent
        }
      })
    });
    results.balance = await balanceResult.json();
  }
  
  // Identity verification
  if (documents.identity) {
    const identityResult = await fetch('https://api.realcap.app/verify', {
      method: 'POST',
      headers: {
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        verificationType: 'rental_identity',
        applicationId: applicationId,
        screenshot: documents.identity,
        metadata: {
          declaredName: documents.declaredName,
          declaredPhone: documents.declaredPhone
        }
      })
    });
    results.identity = await identityResult.json();
  }
  
  // Comprehensive risk assessment
  const overallRisk = calculateOverallRisk(results);
  
  return { results, overallRisk };
}

function calculateOverallRisk(results) {
  let riskScore = 0;
  
  if (results.income?.status === 'failed') riskScore += 40;
  if (results.balance?.status === 'failed') riskScore += 30;
  if (results.identity?.status === 'failed') riskScore += 50;
  
  if (riskScore >= 50) return 'high';
  if (riskScore >= 20) return 'medium';
  return 'low';
}
```

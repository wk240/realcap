**防AI伪造 · 可验真 · 安全证明**

## RealCap 是什么？

RealCap 是一款**可信截图工具**，能够截取带有来源证明的真实、可验证截图。与普通截图不同，RealCap 截图包含加密证据，证明截图来自真实的应用窗口——而非由AI生成或篡改。

在AI生成虚假截图泛滥的时代，RealCap 提供**截图真实性验证**，防范欺诈、虚假信息和证据篡改。

---

## 核心能力

### 1. 可信截图 - 进程验证

RealCap 在截图时检测实际窗口进程，创建**来源证明**：

- 截图来自哪个应用（微信、QQ、浏览器等）
- 窗口标题和进程ID
- 时间戳与加密哈希

**适用场景：**
- 法律证据保全
- 金融交易凭证
- 通话记录验证
- 审计追踪记录

### 2. 直传云端验证

RealCap 使用**可信技术流程**保证图片真实性——无第三方介入：

- **直传流程** - 截图直接从截取 → 云端存储
- **进程验证** - RealCap 在截图时检测来源窗口进程
- **云端查验** - 验证者直接从云端服务器检查图片

**为什么重要：**
与AI检测算法（可能被欺骗）不同，RealCap 的真实性保证来自于整个截取-验证流程都在 RealCap 控制之下。图片不经第三方，确保从截取到验证的完整链路。

### 3. 云端验真

每张RealCap截图生成**验真链接**，任何人可验证其真实性：

- **云端存储** - 截图上传并生成加密哈希
- **验真页面** - 公开URL显示截图元数据和真实性状态
- **可分享证明** - 发送验真链接给第三方进行独立验证

**验证内容包含：**
- 原始截图时间
- 来源应用信息
- 哈希验证（确认图片未修改）
- 进程验证状态

## 工作原理

```
截图 → 进程检测 → 哈希生成 → 云端上传 → 验真链接
```

1. **截图** - 用户通过RealCap客户端截图
2. **进程检测** - RealCap识别来源应用窗口
3. **哈希生成** - 创建图片的加密哈希值
4. **云端上传** - 截图和元数据直接上传至验真服务器
5. **验真链接** - 生成唯一URL用于真实性验证

## 技术细节

### 进程检测技术

RealCap使用系统级API识别活动窗口：
- Windows: Win32 API 窗口枚举
- macOS: Cocoa NSWindow API
- 采集: 进程ID、窗口标题、应用名称

### 哈希验证

每张截图使用SHA-256哈希：
- 哈希与截图元数据一起存储
- 验真页面按需计算哈希
- 任何修改都会改变哈希 → 验证失败

### 直传云端验证流程

RealCap 的真实性保证来自于流程控制：
- 截图直接从截取 → 云端（无第三方介入）
- 验证者直接从云端服务器检查图片
- 整个流程保持完整链路
- 哈希验证确认图片完整性
